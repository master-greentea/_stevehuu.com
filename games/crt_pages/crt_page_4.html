UI/UX Design

The entire game is playable with keyboard only. It's designed this way to fully emulate the retro computer theme. To do so, UI is a completely custom system where it fully utilizes Unity's EventSystem.

The UI buttons are simply presented as a list of text. Selecting a button adds a ">" to the text which signifies selection. It's clean, slick, and intuitive.
