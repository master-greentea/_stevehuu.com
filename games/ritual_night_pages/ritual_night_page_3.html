Frontend

I was responsible for creating all of the in-game front-end UI elements. The process: I get cool mock-ups from our graphic designer. I re-create it responsively in Unity UI. I setup all the references and dependencies so that programmers have the right functions to call. I animate them. I localize all the texts. Then I pass it to the programmers.
